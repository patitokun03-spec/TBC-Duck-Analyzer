❓ How to get the API Keys

1. Log in to Warcraft Logs Profile.

2. Scroll to the bottom of the page to the API Clients section.

3. Create a new client (V2) called "TBC Duck Analyzer".

4. Copy the Client ID (36 characters) and the Client Secret (40 characters).

5. Paste them in the Settings ⚙️ menu on this page.
